# auto-dealership-analysis
🚗 Consumer Purchasing Behaviour Analysis – Austo Automobile
📌 Overview

Exploratory analysis of customer data to understand factors influencing car type and price selection, and derive business insights.

🎯 Objective

Identify how demographic, financial, and lifestyle factors impact:
Car type (Hatchback, Sedan, SUV)
Price segment

📊 Dataset

Customer-level data including:

1. Age, Gender, Marital Status
2. Profession, Dependents, Partner Working
3. Salary, Partner Salary, Loan Status
4. Car Type (Make), Price

🔍 Key Insights

1. Age is the strongest driver of car price
2. Income drives upgrade path across segments
3. Sedans are consistently preferred
4. Dual-income households show stronger spend correlation
5. Loan status has minimal impact on spending

💡 Recommendations
* Target 40+ customers for premium segments
* Position sedans as default upgrade choice
* Focus on value-based messaging over financing
* Use income-based segmentation for targeting

🛠️ Tools

Python (Pandas, NumPy), Seaborn, Matplotlib, Jupyter Notebook
